# Dispositions des panneaux (Layouts)

Au lieu de redimensionner manuellement les panneaux, vous pouvez utiliser les dispositions intégrées pour organiser rapidement votre fenêtre.

## Expérimenter les dispositions

Pour les voir en action, créons trois panneaux :
1. Ouvrez un nouveau panneau vertical : `PREFIX` + `%`
2. Divisez ce nouveau panneau horizontalement : `PREFIX` + `"`

Maintenant, essayez les cinq dispositions principales de tmux :

* **Even Horizontal** (Horizontal équilibré) : `PREFIX` + `Meta` + `1`
  (Tous les panneaux côte à côte, de largeur égale)
* **Even Vertical** (Vertical équilibré) : `PREFIX` + `Meta` + `2`
  (Tous les panneaux empilés verticalement, de hauteur égale)
* **Main Horizontal** (Horizontal principal) : `PREFIX` + `Meta` + `3`
  (Un grand panneau en haut, les autres divisés en bas)
* **Main Vertical** (Vertical principal) : `PREFIX` + `Meta` + `4`
  (Un grand panneau à gauche, les autres divisés à droite)
* **Tiled** (Tuilé) : `PREFIX` + `Meta` + `5`
  (Panneaux disposés en grille)

### Faire défiler les dispositions
Vous pouvez également faire défiler toutes les dispositions disponibles en appuyant sur :
`PREFIX` + `Espace`

Essayez d'ajouter d'autres panneaux et de faire défiler les dispositions pour voir comment tmux les gère !
