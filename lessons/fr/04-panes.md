# Travailler avec les panneaux

Les panneaux vous permettent de diviser une seule fenêtre en plusieurs sections. C'est idéal pour surveiller des logs, lancer des tests ou gérer git tout en codant.

## Diviser la fenêtre

* **Division verticale** : Appuyez sur `PREFIX` + `%`
* **Division horizontale** : Appuyez sur `PREFIX` + `"` (guillemet double)

Essayez de diviser votre fenêtre maintenant. Le nouveau panneau sera automatiquement activé.

## Naviguer entre les panneaux

* **Touches fléchées** : Appuyez sur `PREFIX` + `Touche fléchée` (Gauche, Droite, Haut, Bas).
* **Dernier panneau** : Appuyez sur `PREFIX` + `;` (point-virgule) pour basculer entre les deux derniers panneaux.
* **Passer au suivant** : Appuyez sur `PREFIX` + `o`.
* **Identifier les panneaux** : Appuyez sur `PREFIX` + `q`. Des numéros apparaîtront brièvement sur chaque panneau. Tapez le numéro pour y accéder.

## Organiser les panneaux

### Déplacer les panneaux
* **Échanger avec le précédent** : `PREFIX` + `{`
* **Échanger avec le suivant** : `PREFIX` + `}`

### Zoomer
Si vous avez besoin temporairement de plus d'espace :
* **Zoomer/Dézoomer** : `PREFIX` + `z`. Cela agrandit le panneau actuel pour qu'il occupe toute la fenêtre. Appuyez à nouveau pour revenir à la vue divisée.

### Redimensionner
Vous pouvez ajuster précisément la taille de vos panneaux :
* **Redimensionner de 5 cellules** : `PREFIX` + `Meta` + `Touche fléchée`
* **Redimensionner de 1 cellule** : `PREFIX` + `Ctrl` + `Touche fléchée`

## Fermer des panneaux

* **Quitter le shell** : Tapez `exit` ou appuyez sur `CTRL-d`.
* **Fermer le panneau** : Appuyez sur `PREFIX` + `x` (demande confirmation).
