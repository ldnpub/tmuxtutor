# tmuxtutor (Modernized)

A brief, interactive tutorial for the tmux terminal multiplexer, inspired by `vimtutor`.

## Credits & Origin

This project is a modernized fork of the original [tmuxtutor](https://github.com/perlpunk/tmuxtutor) created by **perlpunk**. 
The original version used Perl and Swim; this version aims to be a standalone, interactive Bash script to provide a better "learning by doing" experience.

A big thank you to **perlpunk** for the original content and pedagogical structure.

## How to use

Simply run the main script:

```bash
./tmuxtutor.sh
```

## Features (Modern version)
- **Interactive validation**: The tutorial checks if you actually performed the tmux commands.
- **Split-screen learning**: Instructions in one pane, practice area in another.
- **No dependencies**: Only requires `tmux` and `bash`.
